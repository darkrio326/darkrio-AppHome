AutoNotice Privacy Policy

隐私政策

AutoNotice 0.2 提供天气提醒、到期提醒与 Developer Webhook。本政策说明精确位置、设备标识、用户提交内容和加密到期内容如何被处理。

适用产品：AutoNotice 最后更新：2026-07-15 无需账号

一、我们是谁

AutoNotice 是 darkrio Studio 提供的轻量自动提醒工具。当前 0.2 版本提供 Weather Notice、Expiry Notice、Developer Webhook、统一通知历史和可选支持开发者 IAP。

本政策适用于 AutoNotice App 以及 app.darkrio326.top 上与 AutoNotice 直接相关的支持页面。

二、我们处理哪些信息

AutoNotice 不要求注册账号，也不会主动收集姓名、手机号、身份证号等个人身份信息。

根据你主动使用的功能，AutoNotice 可能处理以下信息：

  • 精确位置：经纬度、国家和地区、城市或区县名称及时区。
  • 设备标识：稳定 Device ID、APNs device token、Bundle ID、App 版本和 APNs environment。
  • 天气模块规则、概览、触发证据、通知历史和错误状态。
  • 到期时间与重复规则等 metadata，以及设备端加密后的标题和备注 envelope。
  • 通过 Developer Webhook 主动提交的通知标题、正文和可选 payload。
  • 可选支持型 IAP 的 StoreKit 商品状态、购买结果和本机记录。

App Store 隐私标签将精确位置、设备 ID 和其他用户内容标记为与用户关联，因为它们会关联到用于投递的稳定设备身份；这些数据不用于跟踪。

我们不出售个人数据，不投放广告，也不建立跨 App 或网站画像。

三、位置与天气数据

  • 当前位置用于更新所在区县、天气网格和提醒规则。
  • 手动选择地点使用 Common API 的国家和地区、城市及区县基础目录。
  • WeatherKit 提供基础天气，和风天气补充中国官方预警与分钟级降水，RainViewer 提供雷达瓦片。
  • 不同来源可能具有不同更新时间和空间分辨率，App 会展示相应来源说明。
  • AutoNotice 不提供社交定位、实时轨迹分享或广告定位能力。

四、设备、通知与 Notice Engine

AutoNotice Worker 运行在 Cloudflare 基础设施上，负责设备注册、通用 Notice Rule、天气触发、Notice Event、APNs 投递、通知历史和测试通知。

  • Xcode Debug 使用 APNs development；TestFlight 与 App Store 使用 APNs production。
  • 官方天气预警可能使用 iOS 时效性通知。
  • 通知只用于你启用或主动触发的提醒，不包含广告。
  • 你可以在 iOS 系统设置中关闭通知权限。

五、到期提醒与端侧加密

  • Expiry Notice 优先由 iOS 在本机安排通知。
  • 标题与备注在设备端使用 Keychain 管理的密钥加密后再同步；服务端无法读取或解析明文个人内容。
  • 到期时间、提前量、重复规则和来源关系可能作为必要调度 metadata 同步。

六、Developer Webhook

  • Webhook 只投递你或你的 Connector 主动提交的标准通知。
  • 明文 token 与设备鉴权 secret 只保存在本机 Keychain；Worker 只保存不可逆 hash 和短前缀。
  • Token 泄露后应在 App 内重新签发并停用旧凭据。

七、支持型 IAP

支持型 consumable IAP 由 Apple App Store 处理，AutoNotice 不接触银行卡号或完整付款信息。

  • 支持型 IAP 不解锁额外功能。
  • 不购买也可以使用天气、到期提醒、Webhook、历史和设置。

八、数据保留与删除

  • 本机设置、规则、位置偏好、加密密钥和本地通知安排由你在设备上管理。
  • 服务端只在模块运行、投递、历史审计和防滥用所需范围内保留相关状态。
  • 删除 App 可能移除本机副本和本地通知安排，但不会自动向所有远程服务发出删除请求。
  • 如需删除服务端关联状态，可通过支持邮箱联系开发者。

九、第三方与基础设施服务

  • Apple：WeatherKit、APNs、StoreKit、App Store 和系统权限。
  • Cloudflare：官网托管、AutoNotice Worker、D1 和相关提醒基础设施。
  • 和风天气：官方天气预警与分钟级降水补充数据。
  • RainViewer：雷达历史瓦片及元数据。
  • Common API：国家和地区、城市及区县基础资料。

十、未成年人保护

AutoNotice 面向一般工具类使用场景设计。我们不会故意面向未满 14 周岁的未成年人提供服务或收集其个人信息。

十一、政策更新

我们可能会根据产品功能、法律法规或服务配置变化更新本隐私政策。更新后的版本将在本页面发布，并自发布之日起生效。
